About the Project

This interactive web map application visualizes riparian vegetation and land-cover change along the Middle Rio Grande corridor in New Mexico. It is designed for researchers, land managers, conservation planners, and students who want to explore spatial patterns in riparian habitat over time using an accessible, browser-based interface.

Acknowledgements

This project was built using the following datasets, libraries, and examples:

  • Middle Rio Grande Conservancy District (MRGCD) – land cover and irrigation data
  • Leaflet.js Official Tutorials – interactive slippy map and layer controls
  • D3.js Official Documentation – data-driven SVG charts and visualizations
  • D3.js Observable Gallery – chart examples and D3 technique references
  • DC.js Official Examples – linked/brushable charts built on D3 and Crossfilter
  • Crossfilter.js Documentation – fast multidimensional filtering of large datasets
  • Bootstrap 3 Components Documentation – UI layout, modal, and navbar components
  • W3Schools – Bootstrap Modal Tutorial – reference used for this About modal
