About This Project

About the project

This web map app helps users explore riparian ecosystem patterns in the Middle Rio Grande study area. Users can click polygons to view vegetation cover and area information, and can compare tree, shrub, and herbaceous cover through a choropleth map. It can be used by students, researchers, planners, and others interested in riparian land cover.
